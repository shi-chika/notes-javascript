Boolean判斷

Boolean值有2個分別為 true、 false ，在JavaScript中，任何值都可以轉為Boolean值。

以下為falsy values，會被判斷為false

  • False
  • 0或-0
  • 所有空字串 (包含雙引號 " "、單引號 ' '、` ` )
  • Null
  • Undefined
  • NaN

*除此之外都是truthy，包含0以外的數字、字串、空陣列[ ]、空物件{ }等等


JavaScript 中的coding習慣

  • 變數、函數名稱都用小寫，單字組合用駝峰式或底線命名
  • 運算符號左右加空格，方便閱讀
  • 用分號結束
  • 常數用大寫命名 (用const宣告不能改的值)
  • Class 用大小字母開頭

變數、函數名稱的限制

  • 開頭不能為數字
  • 不用連字號-已作為減法運算使用
  • 不能用保留字
