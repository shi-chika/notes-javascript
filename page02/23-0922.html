─ 基本的變數操作、從 html 取得內容 ─

document . getElementById ( ' keyword ' ) . value


  • alert( ) ; 用來顯示訊息視窗的函數，括號內可以放字串及變數
  • document 是一個 JS 在瀏覽器中內建的「物件」，代表網頁當下的狀態
  • 後面的 . 代表去呼叫這個「物件」的「函數」，或者去存取這個「物件」的「屬性」
  • getElementById (' keyword ') 這函數會去找出網頁中 id 為 keyword 的元素
  • . value 功用就是去存取元素的 value 屬性

                function myFunction()
                {
                  const str = document.getElementById('keyword').value;
                  alert(str+"歡迎");
                }
